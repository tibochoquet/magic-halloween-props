# DEVELOPMENT_RULES.md

## Tech Stack

* Next.js
* Tailwind CSS
* TypeScript
* Framer Motion
* shadcn/ui

## Coding Rules

* use reusable components
* avoid duplicate code
* keep components modular
* maintain clean folder structure
* mobile-first responsive design
* optimize performance
* avoid unnecessary dependencies

## Animation Rules

Animations should feel:

* cinematic
* smooth
* subtle
* premium

Use:

* soft fade-ins
* subtle hover effects
* fog drift animations
* ember particles
* smooth transitions

Avoid:

* chaotic movement
* excessive bouncing
* distracting animations

## Ecommerce UX

Product pages should include:

* cinematic gallery
* immersive descriptions
* product specifications
* trust indicators
* add to cart functionality
* related products

The webshop must always feel:

* trustworthy
* professional
* immersive
* premium
